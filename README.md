# 🎵 Spotify Clone

A Spotify-inspired music player UI built with 
HTML, CSS, and JavaScript as a learning project.

## 📸 Screenshot
![Spotify Clone](screenshot.png)

## 🛠️ Tech Stack
- HTML5
- CSS3  
- JavaScript

## ✨ Features
- Spotify-like UI design
- Music player controls
- Play/Pause functionality
- Song list display
- Responsive design
- Album art display

## 🚀 Live Demo
[🔗 Click here to view](https://yash-raj-jain22.github.io/spotify-clone/)

## 📌 How to Run Locally
1. Clone this repo: git clone https://github.com/yashrjain22/spotify-clone.git
2. 2. Open `index.html` in browser
3. Enjoy! 🎶

- Audio API basics
- Responsive web design
